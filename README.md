# Hyatt Talent Marketplace

A UI prototype for a gig economy platform enabling flexible hospitality work at Hyatt hotels.

**Created by Bravo6 IE EMBA Group**

## Demo

This is a visual prototype for academic presentation - no backend, all data is hardcoded.

## Features

### For Workers
- Browse available shifts at luxury Hyatt hotels
- View shift details (role, time, pay rate)
- Apply for shifts with one click
- Track upcoming and completed shifts
- Build ratings and reputation

### For Hotels (Admin)
- Post new shifts
- Review worker applications
- Accept/reject candidates
- Rate workers after shifts

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- React Router
- React Hot Toast
- Lucide Icons

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Pages

| Page | URL | Description |
|------|-----|-------------|
| Landing | `/` | Hero + value propositions |
| Browse Shifts | `/shifts` | Available shifts with filters |
| Shift Details | `/shifts/:id` | Shift info + apply button |
| Worker Profile | `/profile` | Demo worker profile |
| My Shifts | `/my-shifts` | Upcoming/Pending/Completed |
| Admin Dashboard | `/admin` | Hotel management view |
